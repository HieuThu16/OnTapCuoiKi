### Code xử lý thêm sản phẩm (xulythemsp.php)

```php
<?php
// Kết nối database
$host = "localhost";
$username = "root";
$password = "root";
$database = "BaiThiDB";

$conn = new mysqli($host, $username, $password, $database);
if ($conn->connect_error) {
    die("Kết nối thất bại: " . $conn->connect_error);
}

if (isset($_POST['btnthemsanpham'])) {
    $tensp = $_POST['txttensp'];
    $gia = $_POST['txtgia'];
    $giagiam = $_POST['txtgiagiam'] ?? 0;
    $mota = $_POST['txtareamota'];
    $mansx = $_POST['cbocontysx'];

    // Xử lý upload file
    $hinh = "";
    if (isset($_FILES['f_daidien']) && $_FILES['f_daidien']['error'] == 0) {
        $target_dir = "images/";
        $hinh = basename($_FILES['f_daidien']['name']);
        $target_file = $target_dir . $hinh;

        // Tạo thư mục nếu chưa có
        if (!file_exists($target_dir)) {
            mkdir($target_dir, 0777, true);
        }

        move_uploaded_file($_FILES['f_daidien']['tmp_name'], $target_file);
    }

    // Thêm sản phẩm vào database
    $stmt = $conn->prepare("INSERT INTO SANPHAM (tenSanPham, Gia, giamGia, moTa, Hinh, maNhaSanXuat) VALUES (?, ?, ?, ?, ?, ?)");
    $stmt->bind_param("sddssI", $tensp, $gia, $giagiam, $mota, $hinh, $mansx);

    if ($stmt->execute()) {
        echo "<script>alert('Thêm thành công'); window.location='ThemSP.php';</script>";
    } else {
        echo "<script>alert('Thêm thất bại: " . $stmt->error . "'); window.history.back();</script>";
    }

    $stmt->close();
}

$conn->close();
?>
```

**Ghi chú:** Mã sản phẩm tăng tự động (AUTO_INCREMENT trong MySQL).
